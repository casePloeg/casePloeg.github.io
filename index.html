Reminder Bot for Twitter

Created a bot for Twitter that is capable of replying to followers and sending timely reminders when needed. The bot was created using the Twitter API along with the npm library Twit. The bot is self sustainable and is capable of runing 24/7 without maintenance.

  • Used Node.js to set up a server hosting the bot
  • Designed an algorithm that decides when to send reminders to followers
  • Used Firebase to handle database queries
